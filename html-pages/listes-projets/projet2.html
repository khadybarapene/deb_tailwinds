Fitness Mobile App

Description du projet

Fetnes Mobile App est une application mobile développée en utilisant le framework React Native. Elle permet aux utilisateurs de parcourir une large gamme de produits, d'ajouter des articles à leur panier et de passer des commandes en ligne. L'application offre une interface conviviale et sécurisée pour les transactions, ainsi que des fonctionnalités de gestion des stocks et de suivi des commandes pour les administrateurs.

Technologies utilisées

  • React Native
  • JavaScript
  • Redux
  • Node.js
  • Express
